Setup Instructions
Enter your name,should be less than 7 characters. Then click Start button to launch the game.
For more setup information, please click More Info.
Difficulty Level Setup
To choose the size of board, click Select Difficulty Level. By default it will be using 6x7 board. To set the timer for the game, click Set Time Limit. By default it will not be using timer.
Game Play Instructions
Yelow player will be the first to drop yelow token. Since then each player has to drop his token alternately. The person who firstly gets four tokens connected in horizontal or vertical or diagonal direction is the winner.
If you set the timer, each player will have a limited time to drop a token. If one player does not drop his token within time limit, another player will be the winner.
